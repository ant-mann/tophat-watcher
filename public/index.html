Getting Started

  1. Connect Top Hat and sign in using the browser window that opens.
  2. Refresh courses to pull in the classes Top Hat can see.
  3. Turn off any course you do not want monitored.
  4. Start watching and leave the companion running.

When a question is detected

What to expect
  • A single lightweight background watcher checks your courses one at a time until something new appears.
  • A repeating alert sound and desktop notification get your attention.
  • Top Hat opens only when needed and jumps to the course with the live question.
  • If Top Hat shows an answer box, the cursor is placed there for you.
  • Nothing is typed or submitted automatically.

You should never need to edit files or use JSON.
